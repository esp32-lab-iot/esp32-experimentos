# ESP32 Experimentos

Repositório com exemplos didáticos de ESP32, organizados em ordem crescente de complexidade.

## Estrutura

- `exemplos/`: experimentos práticos
- `materiais/`: materiais de apoio
- `docs/`: imagens e arquivos gerais

## Sequência inicial

1. Blink
2. LED com botão
3. Serial básico
4. Wi-Fi básico
5. HTTP GET
6. Google Sheets
7. DHT11 + Google Sheets
